# TP 1 - Backend

Este parcial abarca los contenidos vistos hasta el momento en la materia Desarrollo Web Backend. Se realizará en grupos de 3 a 5 estudiantes de la misma comisión y se enfocará en la creación de una aplicación web utilizando Node.js y Express, con una estructura organizada de carpetas y persistencia de datos en JSON.

##Objetivos específicos
1. Desarrollar una aplicación web utilizando Node.js y Express.
2. Integrar una base de datos en formato JSON. (No usar MongoDB en este parcial).
3. Aplicar conceptos de vistos en los módulos trabajados.
4. Implementar:
    - rutas dinámicas
    - middleware
    - motor de plantillas Pug
5. Probar el funcionamiento de la aplicación utilizando ThunderClient y capturar al menos una evidencia de prueba.
6. Aplicar buenas prácticas de desarrollo, incluyendo el uso de Programación Orientada a Objetos (POO).

## Objetivos generales
- Desarrollar software a partir de una consigna real.
- Integrar equipos de proyecto para el desarrollo colaborativo.
- Asumir roles dentro del grupo de trabajo.
- Desempeñarse de manera autónoma en el desarrollo de sistemas de baja complejidad.

## Criterios de evaluación
### 1. Funcionalidad
- La aplicación debe cumplir con todos los requisitos solicitados. No es necesario extender el proyecto más allá de lo pedido. Se evaluarán los contenidos correspondientes a los módulos vistos y mapa de contenido.

#### 2. Calidad del código 
El código debe:
- estar correctamente organizado
- seguir buenas prácticas
- utilizar estructura modular
- incluir pruebas documentadas

#### 3. Documentación
El proyecto debe incluir:
- bibliografía utilizada (tutoriales, libros, videos, documentación)
- explicación del funcionamiento del sistema
- roles asignados a cada integrante
- responsabilidades específicas dentro del proyecto

Además, deberán grabar un pequeño video donde cada integrante explique su participación.

## Entrega
